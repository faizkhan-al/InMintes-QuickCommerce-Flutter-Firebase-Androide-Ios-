# InMintes-QuickCommerce-Flutter-Firebase-Androide-Ios-
Project Description: InMinutes "InMinutes" is a Blinkit-inspired Q-Commerce app (groceries in 16-mins) built with Flutter &amp; Firebase. Features a "Hybrid Offline Cart" for data safety and a "Print Store" for document delivery. Note: 1st project; focused on UI. OTP is a simulation. No live AI/IoT used.
